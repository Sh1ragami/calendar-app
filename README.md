# Calendar App

## 📝 Description
簡単なカレンダーアプリです。

## ✨ Features
- 月表示カレンダー
- 前月・翌月への移動
- 今日の日付のハイライト
- シンプルなUI/UX

## 🚀 Technologies
- HTML
- CSS
- JavaScript

## 📦 Installation

```bash
# 1. リポジトリをクローン (GitHubにホストする場合)
# git clone [YOUR_REPOSITORY_URL]
# cd calendar-app

# 2. ファイルをローカルに保存
# このアプリはシンプルなHTML/CSS/JSで構成されるため、
# プロジェクトファイルを任意のディレクトリにダウンロードするだけで動作します。
```

## 💡 Usage

1.  ダウンロードしたプロジェクト内の `index.html` ファイルをウェブブラウザで開きます。
2.  カレンダーが表示され、上部のボタンで前月や翌月に移動できます。
